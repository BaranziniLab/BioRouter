In our previous episode of [goose Flight School](https://www.youtube.com/playlist?list=PLyMFt_U2IX4s1pMaidir5P4lSfjUK6Nzm), our host [Adewale Abati](https://www.linkedin.com/in/acekyd/) showed us how to use biorouter to take a Figma design and transform it into a functional Nuxt application. In this stream he covered the entire process, from initial setup to final implementation, highlighting how biorouter can help developers bridge the gap between design and development.

<!--truncate-->

# How It Works
[Extensions](https://baranzinilab.github.io/BioRouter/docs/getting-started/using-extensions) enhance biorouter's functionality by connecting with your existing tools and workflows. They add new features, access external data resources, and integrate with other systems. Learn how multiple extensions, including Figma and Developer, worked together seamlessly to dramatically accelerate development.

# Live Tutorial: biorouter Builds Live
During the livestream, Adewale demonstrated step-by-step how biorouter handled each development phase, from creating the basic application structure to generating responsive layouts using Tailwind CSS. Adewale also highlighted how biorouter addresses potential limitations as you go, showcasing the powerful balance between the biorouter's automation and developer control.

Throughout the stream, Adewale shared valuable tips to prepare your design for biorouter. His key recommendations include: 

* start with a well-structured Figma design
* use biorouter to make targeted improvements after the initial generation
* fine-tune specific elements as needed
* make sure you thoroughly test for functionality and responsiveness

# Getting Started with biorouter and Figma
Whether you're a designer wanting to rapidly turn concepts into working code or a developer curious about streamlining design implementation, you can download biorouter with its built-in [Developer extension](https://baranzinilab.github.io/BioRouter/docs/getting-started/installation) and add the [Figma extension](https://baranzinilab.github.io/BioRouter/v1/extensions/). 

For step-by-step instructions, check out the [Figma tutorial](/docs/mcp/figma-mcp).
